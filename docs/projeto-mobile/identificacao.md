## Descrição Resumida

Aplicativo mobile para rastreamento de hábitos pessoais, permitindo ao usuário cadastrar, visualizar, editar e remover hábitos. O sistema implementa validações de regras de negócio e gerenciamento de estado em memória.

## Stack Tecnológica

- **Framework**: React Native com Expo (Managed Workflow)
- **Linguagem**: TypeScript
- **Navegação**: Expo Router
- **Estado Global**: Zustand
- **Validação**: Zod + React Hook Form
- **Testes**: Jest + Testing Library
