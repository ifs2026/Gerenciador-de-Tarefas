# Log de Prompts - Habit Tracker

Este documento registra os prompts utilizados para auxiliar no desenvolvimento do projeto com ferramentas de IA.

## Formato de Registro

Para cada prompt utilizado, registre:
1. **Data/Hora**: Quando foi utilizado
2. **Ferramenta**: Qual IA foi utilizada (ChatGPT, Claude, Copilot, etc.)
3. **Objetivo**: O que se pretendia alcançar
4. **Prompt**: O texto exato do prompt
5. **Resultado**: Resumo do que foi gerado/modificado
6. **Avaliação**: Se foi útil, precisou ajustes, etc.
